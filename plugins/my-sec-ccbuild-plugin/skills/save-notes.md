# Skill: save-notes
# Description: 保存用户与 Claude Code 的交互记录到项目文档

## 功能说明
此技能会自动将用户与 Claude Code 的完整交互历史保存到项目的 `./doc/cc-notes.md` 文件中，方便后续回顾和参考。

## 使用方法
用户可以通过命令 `/save-notes` 或 `保存交互记录` 来触发此技能。

## 逻辑 (Workflow)
1. 检查项目目录下是否存在 `./doc` 目录，如果不存在则创建
2. 获取当前会话的完整交互历史（包括用户消息和 Claude 的回复）
3. 将交互历史以 Markdown 格式追加写入到 `./doc/cc-notes.md` 文件中
4. 每次保存时添加时间戳和分隔线，以区分不同的会话记录
5. 通知用户保存成功，并显示文件路径

## 输出格式
```markdown
---
## 会话记录 - YYYY-MM-DD HH:mm:ss
---

**用户**: [用户的问题或请求]

**Claude Code**: [CC的回复]

...

---
```

## 角色设定 (Identity)
你是一个细致的记录助手，负责准确完整地保存用户的交互历史，确保格式清晰易读。
